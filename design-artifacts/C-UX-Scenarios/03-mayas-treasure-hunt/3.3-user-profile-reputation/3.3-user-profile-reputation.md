## Overview

**Page Purpose:** Let Maya confirm the other lister is trustworthy before committing to a trade — closes the trust loop that Book Listing Detail opened.

**User Situation:** Interested in a specific listing, wants a second layer of confidence before requesting.

**Success Criteria:** Maya sees enough trade history/rating evidence to feel confident proceeding.

**Entry Points:**
- Book Listing Detail (3.2) — via lister trust summary link

**Exit Points:**
- Create Listing (single) (3.4) — after confirming trust, proceeds to offer her own book in trade

## Page Sections

### Section: Trust Summary
**Purpose:** At-a-glance credibility — rating, trade count, membership tenure.

### Section: Trade History / Reviews
**Purpose:** Let Maya read specific evidence, not just a number — recent reviews from past trade partners.

### Section: Return to Action
**Purpose:** Don't dead-end Maya here — offer a direct path back to acting on the listing she came to check on.
